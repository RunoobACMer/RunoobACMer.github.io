借用大佬模板
原文

作用范围

单峰函数的最值。

模板

凸函数求极大值：

整型：

while(l+1<r){
int lm=(l+r)>>1,rm=(lm+r)>>1;
if(judge(lm)>judge(rm))
r=rm;
else
l=lm;
}
//答案取 l

小数：

while(l+eps<r){
double lm=(l+r)/2,rm=(lm+r)/2;
if(judge(lm)>judge(rm))
r=rm;
else
l=lm;
}
//答案取 l 或 (l+r)/2 (尽管此时 l 和 r 已经相等，但因为精度问题，取 r 可能会错)
